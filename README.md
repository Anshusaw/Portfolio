# 🚀 3D Portfolio – Interactive Humanoid Experience

> ⚡ A next-generation developer portfolio with **3D visuals**, **humanoid interaction**, and **real-time motion response**

---

## 🌌 Overview

This is not just a portfolio — it's an **immersive experience**.

Built with modern web technologies, this project features a **humanoid robot that reacts to user movement**, creating a futuristic and engaging interface that captures attention instantly.

🧠 Designed to impress recruiters
🎯 Built to stand out from traditional portfolios

---

## 🤖 Key Features

✨ **Humanoid Interaction**

* A 3D humanoid model that dynamically responds to cursor movement
* Creates a lifelike and engaging user experience

🎮 **Motion-Based Responsiveness**

* UI elements react in real-time to user actions
* Smooth animations and transitions

🌐 **Modern UI/UX**

* Clean, minimal, and futuristic design
* Fully responsive across devices

⚡ **Performance Optimized**

* Fast loading with optimized assets
* Smooth rendering using modern frameworks

---

## 🛠️ Tech Stack

* ⚛️ React.js
* ⚡ Vite
* 🎨 Tailwind CSS
* 🧊 Three.js (3D rendering)
* 🌐 HTML5, CSS3, JavaScript

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Anshusaw/Portfolio.git
cd Portfolio
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the project

```bash
npm run dev
```

### 4️⃣ Open in browser

```
http://localhost:5173
```
